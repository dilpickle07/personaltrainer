# Personal trainer web app

This is a simple starter shell for a personal trainer website.

## Planned direction
- Build a web app instead of a desktop program so it works on phone, tablet, and desktop.
- Start with onboarding: age, height, weight, gender, goals, and gym equipment.
- Add future sections for:
  - daily food and workout logging
  - weekly progress summaries
  - mood and energy check-ins
  - progress tracking over time

## Current shell
- index.html: onboarding form plus new daily log and weekly review sections
- styles.css: updated visual layout with navigation and feature cards
- app.js: lightweight starter behavior for the form

## Next steps
1. Define the exact goal options and coaching categories.
2. Decide how daily food and workout entries should be structured.
3. Add a dashboard with charts or progress cards.
4. Connect the form to local storage or a backend later.
